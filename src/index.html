ZeroRPC

So good you'll forget the R

ZeroRPC is a light-weight, reliable and language-agnostic engine for distributed communication. Support for streamed responses - similar to python generators - makes ZeroRPC more than a typical RPC engine. Built-in heartbeats and timeouts detect and recover from failed requests. Introspective capabilities, first-class exceptions and the command-line utility make debugging easy. ZeroRPC powers the infrastructure behind dotCloud.

Installation

  • python
  • node.js
pip install zerorpc
npm install zerorpc
